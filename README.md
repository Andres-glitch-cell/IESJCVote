# 🗳️ IESJCVote - Sistema de Votaciones Digitales

**IESJCVote** es una aplicación web moderna diseñada para gestionar procesos de votación de forma ágil y segura. Este proyecto ha sido desarrollado como **Proyecto de Final de Curso (FCT)** para el **IES Joan Coromines**.

La plataforma permite a los usuarios participar en encuestas y ver resultados en tiempo real, combinando una arquitectura de servidor sólida con una interfaz dinámica e interactiva.

---

## 🚀 Tecnologías Utilizadas

Para este proyecto se ha seleccionado un stack tecnológico que equilibra rendimiento y facilidad de uso:

* **Backend:** PHP 8.x con el framework **Laravel 11**.
* **Frontend:** **JavaScript (ES6+)** para la lógica de cliente y **jQuery** para la manipulación ágil del DOM y peticiones AJAX.
* **Diseño:** **Tailwind CSS** o **Bootstrap** para una interfaz responsiva.
* **Base de Datos:** **SQLite** (Base de datos local en archivo, no requiere privilegios de administrador).
* **Gráficos:** **Chart.js** para la visualización de resultados en vivo.

---

## ✨ Funcionalidades Principales

* ✅ **Creación de Encuestas:** Panel de administración para gestionar preguntas y opciones de forma dinámica.
* ✅ **Votación en Tiempo Real:** Interfaz interactiva que registra votos sin recargar la página mediante AJAX/jQuery.
* ✅ **Visualización de Resultados:** Gráficos estadísticos que se actualizan automáticamente al recibir nuevos datos.
* ✅ **Seguridad:** Validación de datos en el servidor y protección contra votos duplicados.
* ✅ **Diseño Adaptable:** Interfaz optimizada para móviles, tablets y ordenadores.
